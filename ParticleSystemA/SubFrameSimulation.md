# Sub-Frame Simulation

![Full Node Tree](img/ParticleSystemA_FullTree.png)

## Purpose

Elementary particle system to build on.

Features / abilities
- **Subframe** simulation through combinaison of **Simulation Zone** and **Repeat Zone**
- **Gravity**
- **Field(s)**
- **Euler Explicite** simulation type with position update on velocity
- **basic** emission parameters
- Object driven emission with an (Empty as) **Emitter**
